How to Play
The Setup

A standard 52-card deck is shuffled. Nine cards are dealt face-up into nine piles. The remaining 43 cards form the draw pile.

Each Turn
1. Tap a pile to select it. 2. Guess whether the next card drawn will be Higher, Lower, or the Same value as the top card. 3. A card is drawn and placed on that pile. Wrong guess = −1 point.
Card Values

A = 1  ·  2–10 face value  ·  J = 11  ·  Q = 12  ·  K = 13

Scoring
  • Start at 0. Each wrong guess costs 1 point.
  • Finish with 0 for a perfect game.
  • The game ends after all 43 draws.
Options
Hints Shows the probability of each outcome based on cards not yet seen. Sudden Death A wrong guess kills that pile — its card is flipped and it can't be selected again. Must be enabled before the first turn.
